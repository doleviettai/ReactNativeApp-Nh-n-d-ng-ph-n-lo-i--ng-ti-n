Dấu ngoặc đơn (onboarding) = route group — nhóm các file lại nhưng không thêm vào URL. Nên /instruction thay vì /onboarding/instruction. Dùng để share 1 layout swipe chung cho 3 slide.
Điều hướng giữa các màn — dùng router từ expo-router:

import { router } from 'expo-router'

router.push('/choose')  → push màn mới
router.replace('/choose') → thay thế (không back được)
router.back()            → quay lại
Swipe onboarding — dùng FlatList có sẵn:

3 slide Welcome/Instruction/Prepare để trong (onboarding)/_layout.tsx với FlatList horizontal pagingEnabled. Khi swipe xong slide cuối thì gọi router.replace('/choose') để vào app chính.
Không có thư mục src/ — đặt thẳng ở root:

components/  → root
constants/   → root
services/    → root
types/      → root
assets/     → root (đã có sẵn)

Chỉ có app/ là thư mục đặc biệt của expo-router. Phần còn lại đặt ở root bình thường.
Truyền dữ liệu sang màn result — dùng query params:

router.push({ pathname: '/result', params: { imageUri: uri } })

Nhận ở result.tsx:
import { useLocalSearchParams } from 'expo-router'
const { imageUri } = useLocalSearchParams()
